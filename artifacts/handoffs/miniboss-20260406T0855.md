## Key Decisions Made

1. **Daily loop queries pulse.db directly** (not CLI) because `hydra-pulse` CLI lacks a `--period yesterday` option. Python3 sqlite3 module, read-only connection.

2. **Context-limit detection uses prompt_count > 200 + duration > 60min** instead of cumulative token counts, because `total_cache_read` is session-cumulative (millions), not current window size.

3. **Misroute cost delta only counts wasteful misroutes** (actual more expensive than recommended). Cheaper misroutes are tracked separately but don't inflate the cost waste metric.

4. **Unclassified fix commits are not auto-bucketed** into any error category. They contribute to the aggregate fix ratio but don't inflate specific categories.

5. **Hook integration is additive, not new hooks** — preflight-check.sh and behavioral-compliance-gate.sh were extended, no new hook entries in settings.json.

6. **CB metrics are cumulative counters** — the daily loop stores raw counter snapshots in JSON so the next day can compute deltas.
